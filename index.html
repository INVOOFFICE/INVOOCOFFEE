Sync multi-appareils — Supabase

Synchronisez le stock (articles) et l’historique (factures) entre plusieurs navigateurs ou appareils. Optionnel : vous pouvez laisser désactivé.

  1. Ouvrez Supabase et créez un projet (gratuit).
  2. Allez dans SQL Editor, collez le script ci-dessous et exécutez-le une seule fois.
  3. Récupérez l’URL du projet (Settings → API → Project URL) et la clé anon public (pas service_role).
  4. Collez-les ci-dessous, puis Connecter & synchroniser.
Ouvrir Supabase → Project Settings → API
Utilisez la clé anon public — jamais la clé service_role (elle contourne la sécurité).
À cette heure, une synchronisation vers Supabase est lancée automatiquement (même logique que « Synchroniser maintenant »). L’application doit rester ouverte dans un onglet à ce moment-là.
